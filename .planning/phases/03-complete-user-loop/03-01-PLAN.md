**Copy to clipboard (OUT-01):**
- In _CompleteView (when text is non-empty), add a prominent "Copy Text" FilledButton.icon with Icons.copy as the FIRST action button, above the existing "Try Another" button.
- On tap: call `_copyToClipboard(context, result.text)` helper function:
  ```dart
  Future<void> _copyToClipboard(BuildContext context, String text) async {
    await Clipboard.setData(ClipboardData(text: text));
    if (!context.mounted) return;
    ScaffoldMessenger.of(context)
      ..clearSnackBars()
      ..showSnackBar(
        const SnackBar(
          content: Row(
            children: [
              Icon(Icons.check_circle, color: Colors.white, size: 20),
              SizedBox(width: 8),
              Text('Text copied to clipboard'),
            ],
          ),
          duration: Duration(seconds: 2),
          behavior: SnackBarBehavior.floating,
        ),
      );
  }
  ```
- Import `package:flutter/services.dart` for Clipboard.
- Keep the existing metadata row (timer, dimensions, debug toggle) and debug panel.
- Rename "Try Another" to "Try Different Image" and make it an OutlinedButton (visual hierarchy: Copy is primary action, Try Different is secondary).
